# dt-fellowship-reflection-tree
**Candidate submission for the DeepThought BA/DS Fellowship**

A deterministic, LLM-free daily reflection agent built on a branching decision tree. The tree encodes three psychological axes — Agency, Orientation, and Radius — into a structured 15-question conversation that walks an employee through honest self-reflection at end of day.

---

## Repository Structure

```
/tree/
  reflection-tree.json     ← Part A: The full tree data (60+ nodes)

/agent/
  index.html               ← Part B: Single-file web agent (open in browser)

/transcripts/
  persona-1-transcript.md  ← Victim / Entitled / Self-Centric path
  persona-2-transcript.md  ← Victor / Contributing / Altrocentric path (+ comparison table)

write-up.md                ← Design rationale and psychological sources (2 pages)
README.md                  ← This file
```
